# 📊 Stock Portfolio Tracker (Python CLI Project)

## 📌 Overview
This is a simple Python command-line project that helps users track their stock investments using predefined stock prices.

The program calculates total investment based on user input and generates a summary report saved in a text file.

---

## 🚀 Features
- Predefined stock price list
- User input for stock selection and quantity
- Automatic investment calculation
- Portfolio summary display
- Saves results to `portfolio.txt`
- Input validation for better reliability

---

## 🛠️ Technologies Used
- Python 3
